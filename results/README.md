Results folder

Purpose and required deliverable:

- The person assigned should make a final Markdown document of their findings (include methods, key results, plots, and short discussion).

What to include in the final Markdown:

- Brief description of methods and models used.
- Summary tables and key plots (link to files in `outputs/graphs`).
- Discussion of findings, failure modes, and future work.

Naming:

- Save the final document as `name_task_final_results.md` in this folder.
